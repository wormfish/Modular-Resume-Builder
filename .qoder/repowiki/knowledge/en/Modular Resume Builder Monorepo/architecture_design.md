The root `package.json` is a private workspace that owns shared dev tooling (Vite, React, concurrently) and exposes two sibling processes: `npm run server` boots the Express/Mongoose backend from `server/index.js`, while `npm run dev` starts the Vite React dev server. `vite.config.js` proxies every `/api/*` request to `http://localhost:3001`, so the frontend calls the backend via the same origin in development; production builds are separate static assets consumed by the backend or deployed independently. The two children communicate exclusively over REST on this `/api` path — there is no shared code between them.
